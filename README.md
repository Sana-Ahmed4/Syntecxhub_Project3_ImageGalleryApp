#  Image Gallery App

A **responsive and minimal image gallery** built with **HTML** and **CSS**, featuring a grid layout, hover effects, and captions. Perfect for showcasing products, photos, or artwork.  

---

##  Features

- **Responsive Grid Layout**  
  Automatically adjusts columns based on screen size using CSS Grid with `auto-fit` and `minmax`.  

- **Hover Effects**  
  Images zoom and fade smoothly when hovered over for an interactive experience.  

- **Image Captions**  
  Each image displays its name in the center when hovered.  

- **Clean & Minimal Design**  
  Rounded images, semi-transparent caption overlays, and a simple color scheme.  

- **Mobile-Friendly**  
  Fully responsive layout with media queries for smaller screens.  

---

##  Technologies Used

- HTML5  
- CSS3 (Grid, Transforms, Transitions, Media Queries)  

---
---

##  How to Use

1. Clone or download this repository.   
2. Open `index.html` in a modern browser.  
3. Hover over images to see zoom and caption effects.  
4. Resize the browser or open on mobile to see the responsive layout.  

---

##  Customization

- **Add/Remove Images**: Add new `<div class="item">` blocks with `<img>` inside `.gallery`.  
- **Change Hover Effects**: Modify `.item:hover img` in `index.css` for zoom, opacity, or transition speed.  
- **Change Caption Style**: Customize `.caption` background, font, color, or animation.
